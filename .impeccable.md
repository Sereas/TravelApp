## Design Context

### Users
Solo travelers and couples/small groups planning trips together. The context is personal trip planning — researching destinations, organizing locations, building day-by-day itineraries, and sharing plans with travel companions. Users are in "dreaming and organizing" mode, not transactional booking mode.

### Brand Personality
**Warm, anticipatory, crafted.**

The interface should feel like a well-made travel journal that thinks ahead — not a productivity tool, not a booking engine, not a social feed. It's opinionated about quality and effortless in use.

### Aesthetic Direction
- **Visual tone:** Warm minimal — clean surfaces with tactile depth on interaction. Brown/green earth tones, not cold blues or neon accents.
- **References:** Wanderlog (visual trip planning with maps), Google Trips (simple, smart, card-based), Apple Maps Guides (native-feeling, fluid, map-centric warmth).
- **Anti-references:** TripAdvisor/Booking.com (dense, ad-heavy portals), Jira/Linear (productivity tool energy), Pinterest (pretty but unstructured), generic SaaS dashboards (tables, filters, charts).
- **Theme:** Light mode primary. Warm neutral base with forest green brand accent and rich brown CTAs. Dark mode supported.
- **Font:** Inter (established in codebase).
- **Motion:** Functional, not decorative. Frequency-based: high-frequency actions get zero animation, rare moments get delight. motion library for springs/gestures, CSS transitions for simple states.

### Design Principles
1. **Living planning surface, not a dashboard.** Every component should feel designed for trip planning, not assembled from generic UI parts.
2. **One step ahead.** The UI anticipates the next action — suggest, reveal, connect. Never wait passively.
3. **Fluid, not fragmented.** Places, itinerary, and map feel like one connected surface. No hard tab boundaries. Inline editing over modals.
4. **Speed is intelligence.** Optimistic updates, instant feedback, no loading spinners when avoidable. Slow UI feels broken.
5. **Crafted warmth over clinical polish.** Subtle grain, stamp badges, ticket-card perforations — tactile details that feel handmade, not generated.
